# Iterative Number Classification System


A simple number analysis utility that classifies integers based on the sum of their divisors using an iterative algorithm.

This project demonstrates basic algorithm design, iteration, and number classification concepts by evaluating whether a number is perfect, abundant, or deficient.


## Overview

Number classification is a common programming exercise used to demonstrate algorithmic reasoning and control flow.

This program analyzes a number by iteratively calculating the sum of its proper divisors and comparing that sum to the original number.

The result determines the classification:

Perfect Number – the sum of its proper divisors equals the number

## Concepts Demonstrated

- Separation of concerns (fixed range vs. user-driven range)
- Code reuse via method calls
- Handling edge cases with appropriate data types

  ## Example
  ***Input:***
  ```
  Enter upper limit to search for perfect number:
  625,000
  ```
  ***Output:***

  28 is a perfect number.
  496 is a perfect number.
